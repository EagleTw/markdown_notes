# [Tips] Terminal Shortcuts

###### tags: `SNPS`, `Programming`, `Tips`

### Typing commands

| Commands | Description     |
| -------- | --------------- |
| <ctrl>+a | Go to the front |
| <ctrl>+e | Go to the back  |
| <ctrl>+w | Delete a word   |

### Flip Page

| Commands | Description     |
| -------- | --------------- |
| <ctrl>+f | 1 page Forward  |
| <ctrl>+b | 1 page Backward |
| <ctrl>+d | 1/2 page UP     |
| <ctrl>+u | 1/2 page DOWN   |
